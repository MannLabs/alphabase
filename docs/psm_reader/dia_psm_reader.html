DIA PSM reader

This module includes:

  • DiannReader: DIA-NN PSM (.tsv)
  • SpectronautReader: Spectronaut generated library (.tsv or .csv)
  • SpectronautReportReader: Spectronaut result report (.tsv or .csv)
  • SwathReader: Swath library (.tsv or .csv)

As we know so far, all DIA search engines have similar tabular files to MaxQaunt, so here, all DIAReaders are inherited from MaxQauntReader.
