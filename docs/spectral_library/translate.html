create_modified_sequence

 create_modified_sequence (df_items:Tuple, translate_mod_dict:dict=None,
                           mod_sep='()', nterm='_', cterm='_')

Translate (sequence, mods, mod_sites) into a modified sequence. Used by df.apply(). For example, ('ABCDEFG','Mod1@A;Mod2@E','1;5')->_A[Mod1@A]BCDE[Mod2@E]FG_.

Type Default Details
df_items typing.Tuple must be (‘sequence’,‘mods’,‘mod_sites’)
translate_mod_dict dict None A dict to map alpha modification names to other software
mod_sep str () ‘[]’ or ‘()’, default ‘()’
nterm str _
cterm str _
